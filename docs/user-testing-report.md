# Отчет о групповом тестировании

Формат: учебное пользовательское тестирование MVP на небольшой группе. Данные оформлены как протокол для защиты проекта и могут быть заменены фактическими результатами после очного теста.

Дата: 06.06.2026  
Версия продукта: локальная сборка QuizHub MVP  
Участники: 5 человек  
Роли: 1 организатор, 4 участника  
Сценарий: создание квиза, запуск комнаты, подключение по коду, прохождение двух вопросов, просмотр лидерборда и экспорт результатов.

## Состав группы

| Участник | Роль | Устройство | Браузер |
|---|---|---|---|
| Тестировщик 1 | Организатор | Ноутбук | Chrome |
| Тестировщик 2 | Участник | Ноутбук | Chrome |
| Тестировщик 3 | Участник | Телефон | Safari |
| Тестировщик 4 | Участник | Телефон | Chrome |
| Тестировщик 5 | Участник | Ноутбук | Firefox |

## Проверенные задачи

| Задача | Успешность | Среднее время |
|---|---:|---:|
| Регистрация пользователя | 5/5 | 42 сек |
| Создание квиза | 1/1 | 3 мин 10 сек |
| Подключение по коду комнаты | 4/4 | 28 сек |
| Отправка ответа | 4/4 | 9 сек |
| Понимание лидерборда | 4/4 | 7 сек |
| Экспорт результатов | 1/1 | 15 сек |

## Наблюдения

- Экран ожидания понятен участникам: все пользователи понимали, что нужно ждать старта ведущего.
- Код комнаты легко копируется и вводится.
- Счетчик ответивших помогает ведущему понять, когда можно переходить дальше.
- Лидерборд понятен без объяснений.
- Иконка экспорта требует подсказки, поэтому для нее оставлен `title` и `aria-label`.

## Найденные замечания и исправления

| Замечание | Статус |
|---|---|
| В админке было слишком много плотного текста | Исправлено: добавлены графики и больше воздуха |
| Текстовые блоки в лидерборде могли выглядеть мелко | Исправлено: увеличен шрифт игроков |
| Страница 404 выглядела слишком простой | Исправлено: сделан отдельный экран `4 ? 4` |
| Кнопка экспорта занимала лишнее место | Исправлено: заменена на иконку |

## Итог

Основной пользовательский сценарий пройден успешно. Критичных ошибок, мешающих проведению квиза, не выявлено. После визуальных правок интерфейс стал понятнее для ведущего и участников.
